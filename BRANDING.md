# CODAI Branding Guidelines

## Logo and Visual Identity

The CODAI logo, icon, and visual design elements are proprietary assets and are protected by copyright.

### Logo Files
- `icon.png` - Main CODAI logo
- `favicon.png` / `favicon.ico` - Browser favicon
- Any other visual assets in this repository

### Usage Rights

#### ✅ Permitted Uses:
- Using the logo when running an unmodified instance of CODAI
- Including the logo in articles/documentation about CODAI
- Displaying the logo to indicate compatibility with CODAI

#### ❌ Restricted Uses:
- Creating derivative logos or modifying the CODAI logo
- Using the logo for different/competing products
- Implying endorsement without permission
- Removing or obscuring the logo in CODAI instances

### Brand Elements

**Name**: CODAI  
**Tagline**: Evolved Intelligence  
**Positioning**: Turn ideas into production-ready apps and solutions. Zero code required.

### Attribution Requirements

When using CODAI or creating derivative works:

1. **Maintain CODAI branding** visible in the application
2. **Include attribution**: "Originally created by Arian Rudd"
3. **Link to**: https://codai.ai when practical

### Modifications

If you modify CODAI for your own use:
- You may add "Modified by [Your Name]"
- You must keep "Based on CODAI by Arian Rudd"
- Do not remove or replace the CODAI logo without permission

### Questions

For questions about branding usage or permission requests:
- Website: https://codai.ai
- Creator: [@AriRudd](https://x.com/AriRudd)

*These guidelines help maintain brand consistency and recognition while respecting the open-source nature of the project.*
